About

Selko Digital HTML Library is a reference collection of accessible HTML components. The goal of the library is to be clear, easy to use and concise — each component is explained in simple terms with practical, ready-to-use examples.

The library shows one way to implement accessible components. It is not the only correct approach, and implementations may vary depending on context and project requirements. Use the examples as a starting point and adapt them to your needs.

How to use

  1. Browse components from the Components page. Use the category filter to narrow down the list.
  2. Open a component page to read about its accessibility features and see a live example.
  3. Copy the code from the HTML, CSS and Javascript tabs, or click Open in CodePen to experiment with it directly in the browser.
  4. Adapt the code to fit your project — adjust colours, sizes and structure as needed.

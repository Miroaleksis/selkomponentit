About

This site is a quick-reference library of accessible HTML. It serves as a clear, easily scannable guide for developers, designers and other interested parties looking for practical help implementing universally usable components. Look up examples on how a component should be implemented, check which elements and attributes it requires, or copy working code directly into your project.

The components in this library follow the Web Content Accessibility Guidelines (WCAG) 2.2. The examples are designed to meet Level AA conformance, which is the standard required by most accessibility regulations and policies.

This library is not an official specification, a comprehensive accessibility guide or a substitute for understanding the underlying standards. It assumes at least a basic familiarity with accessibility concepts. It is a practical quick-reference tool, designed to help users reach working, accessible code without unnecessary friction.

The library shows one way to implement accessible components. The examples aim to be easy to follow, reliable across browsers and programmatically correct. It is not the only correct approach however and implementations may vary depending on context and project requirements. Use the examples as a starting point or as a framework, not as a rigid specification.

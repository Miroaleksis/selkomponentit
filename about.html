About

Selko Digital HTML Library is a quick-reference library of accessible HTML components. It is designed to be easy to scan, read and use. Look up how a component should be implemented, check which ARIA attributes it requires, or copy working code directly into your project.

Browse components from the Components page and open any component to read about its accessibility features and see a live example. The Attributes page lists the most commonly used ARIA roles and attributes as a quick reference. Copy the code from the HTML, CSS and Javascript tabs, or click Open in CodePen to experiment directly in the browser.

The library shows one way to implement accessible components. It is not the only correct approach and implementations may vary depending on context and project requirements. Use the examples as a starting point, not as a rigid specification.

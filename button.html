Button

A button triggers an action when activated. A toggle button extends this by maintaining a pressed state, communicating whether a setting is currently active or inactive.

Accessibility Features

= Required = Recommended
  • type="button" prevents accidental form submission when the button is inside a form element
  • Button text or aria-label must clearly describe the action the button performs
  • aria-pressed communicates the current toggle state to assistive technology
  • The visual appearance reflects the aria-pressed value so sighted users see the same state as screen reader users hear

Button

A button triggers an action when activated, such as submitting a form, opening a dialog or toggling a setting. Buttons can be momentary or stateful depending on their purpose.

Buttons and links are often confused but serve different purposes. A button performs an action on the current page. A link navigates the user to a new location. See the Link page for more information.

Accessibility Features

= Required = Recommended

Toggle Button

A toggle button is a button that maintains a pressed or unpressed state between activations, such as a play/pause control or a mute button. aria-pressed communicates the current state to screen readers. The visual appearance should reflect the aria-pressed value so sighted users see the same state that screen readers announce.

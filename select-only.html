Select-Only

A Select-Only combobox is an input widget that, when activated, displays a popup listbox containing predefined options. The user can select one option from the list. It does not allow freeform text entry — see Dropdown with Search for that variant.

Select-only comboboxes can be used whenever the user picks one option from a known, limited set of values, such as a country, sort order or category.

The recommended implementation is the native <select> element, which provides keyboard support, screen reader announcements and platform-native rendering for free. A custom implementation may be necessary when the visual design cannot be achieved with the native element, since <select> styling is limited across browsers.

Accessibility Features

= Required = Recommended
  • Expands and collapses with Space and Enter
  • Options are navigable with ↑ and ↓ arrow keys
  • Pressing Esc closes the dropdown and returns focus to the trigger
  • The expanded/collapsed state is conveyed with aria-expanded attribute
  • The widget role is conveyed with role="combobox" on the trigger element

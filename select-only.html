Select-Only

A Select-Only combobox is an input widget that, when activated, displays a dropdown list of predefined options. The user can select one option from the list. It does not allow freeform text entry.

Select-only comboboxes can be used whenever the user picks one option from a known, limited set of values, such as a country, sort order or category.

Accessibility Features

= Required = Recommended
  • Expands and collapses with Space and Enter
  • Options are navigable with ↑ and ↓ arrow keys
  • Pressing Esc closes the dropdown and returns focus to the trigger
  • Home and End move focus to the first and last option

Select Element

The native <select> element provides keyboard support, screen reader announcements and platform-native rendering for free. Its styling is limited across browsers.

Custom Select-Only

A custom implementation may be necessary when the visual design cannot be achieved with the native <select> element. It requires explicit ARIA to communicate widget semantics: role="combobox" on the trigger element identifies the widget, and aria-expanded communicates the expanded or collapsed state to screen readers.

Card

A card is a contained unit that groups related information and actions about a single subject. Cards are typically used in grids or lists to present multiple items in a consistent, scannable layout.

Cards can be used to present a collection of similar items where each item has a consistent set of details that can be scanned at a glance. They are less suitable for displaying long-form content or complex data, where a list or table communicates the structure more clearly.

Accessibility Features

= Required = Recommended
  • The card itself must not be a link. If the entire card should be clickable, the "Read more" link's click area can be extended to cover the full card using a ::before pseudo-element.
  • The link uses aria-labelledby="card-title card-link" to combine the heading text with the link text. This ensures the purpose of the link is conveyed to screen readers in case there are multiple cards with identical "Read more" links.
  • The card may use role="group" and aria-roledescription="card" to identify it as a distinct unit to screen readers, with aria-labelledby pointing to the heading

Date Picker

A date picker allows users to select a date from a calendar popup or by typing it directly into a field. It is used in forms where the user needs to specify a particular date, such as a booking, deadline or date of birth.

The native <input type="date"> element is the recommended implementation. It provides a built-in date picker interface, keyboard support and screen reader announcements without any additional scripting.

Accessibility Features

= Required = Recommended
  • The input must have a visible <label> associated via matching for and id attributes
  • type="date" communicates the expected input format to assistive technology and triggers a native date picker on mobile devices
  • Use min and max attributes to constrain the selectable range when applicable
  • Provide a helper text hint that describes the expected date format for users who type the date manually

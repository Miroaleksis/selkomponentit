Input Group

An input group is a set of related form controls grouped together under a shared label.

The recommended implementation is the native <fieldset> and <legend> elements, which group the controls semantically and provide the group label to screen readers automatically.

Accessibility Features

= Required = Recommended
  • <fieldset> groups the related inputs together semantically, and <legend> provides a label for the entire group
  • Each input must have an associated individual <label>, linked via nesting or matching for and id attributes
  • Use type="checkbox" inputs when users can select any number of options from a group. Use type="radio" inputs when only one option can be selected at a time
  • Checkboxes and radio buttons that belong to the same group share a name attribute
  • <form> wraps related fieldsets into a single semantic container. When given an accessible name via aria-label or aria-labelledby, it is exposed as a form landmark that screen reader users can navigate to directly

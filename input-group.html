Input Group

An input group is a set of related form controls grouped together under a shared label. A typical use case is a group of checkboxes where users can select multiple options from a set, such as filtering by category.

The recommended implementation is the native <fieldset> and <legend> elements, which group the controls semantically and provide the group label to assistive technology automatically.

Accessibility Features

= Required = Recommended
  • <fieldset> groups the related inputs together semantically, and <legend> provides a label for the entire group
  • Each input must have an associated individual <label>, linked by nesting or via matching for and id attributes.
  • Inputs that belong to the same group share a name attribute
  • <form> wraps related fieldsets into a single semantic container. When given an accessible name via aria-label or aria-labelledby, it is exposed as a form landmark that assistive technology users can navigate to directly

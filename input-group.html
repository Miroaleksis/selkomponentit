Input Group

An input group is a set of related form controls grouped together under a shared label. Common examples are radio button groups and checkbox groups.

Input groups can be used whenever multiple form controls share a common purpose or label, such as choosing one option from a list or selecting multiple options from a set.

The recommended implementation is the native <fieldset> and <legend> elements, which group the controls semantically and provide the group label to assistive technology automatically.

Accessibility Features

= Required = Recommended
  • <fieldset> groups the related inputs together semantically, and <legend> provides a label for the entire group
  • Each input must have an associated individual <label>, linked by nesting or via matching for and id attributes.
  • Inputs are marked as part of the same group using a shared name attribute

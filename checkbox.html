Checkbox

A checkbox allows users to select or deselect an option independently. A switch is a binary toggle that turns a setting on or off and shares the same underlying input element, with a different role to communicate its purpose.

Accessibility Features

= Required = Recommended
  • <label> associates visible text with the input so screen readers announce it correctly
  • type="checkbox" provides the native checkbox role and communicates checked state without additional ARIA

Switch

A switch is a binary toggle that turns a setting on or off. role="switch" overrides the implicit checkbox role to communicate this on/off nature to screen readers. The native checkbox is visually hidden but remains in the accessibility tree so focus and state are handled by the browser.

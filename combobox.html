Combobox

A combobox is an input control that displays a dropdown list of options. The list appears on command, keeping the interface compact until the user needs to make a selection. A combobox combines the behaviour of several components (hence the name): it expands and collapses like an accordion, its options behave like radio buttons or checkboxes, and it may also contain other input methods like text input.

Select-Only

A simple combobox that allows users to select a single option from a dropdown list.

Multiselect

A combobox that allows users to select multiple options from a dropdown list.

Combobox with Search

A combobox that filters the list of options as the user types.

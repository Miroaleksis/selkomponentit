Combobox

A combobox is an input widget that has an associated dropdown list of options. The list appears on demand, keeping the interface compact until the user needs to make a selection.

Select-Only

A simple dropdown combobox that shows a list of options when the trigger is activated.

Multiselect

A combobox that allows users to select multiple options from a dropdown list.

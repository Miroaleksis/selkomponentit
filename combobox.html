Combobox

A combobox is an input widget that has an associated popup list of options. There are two variants: a simple dropdown and a dropdown with a search field for filtering options.

Dropdown

A simple dropdown combobox that shows a list of options when the trigger is activated.

Dropdown with search

A combobox with an inline text field that filters the list of options as the user types.

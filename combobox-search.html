Combobox with Search

A combobox with search is an input widget that filters a dropdown list of options as the user types. It is suited for longer lists where browsing all options at once would be impractical.

The widget can be configured for single selection, where choosing an option closes the dropdown, or multi selection, where several options can be chosen simultaneously.

Accessibility Features

= Required = Recommended
  • role="combobox" on the input element identifies the widget. aria-expanded and aria-controls communicate state and target to screen readers
  • aria-autocomplete="list" indicates that the displayed options are filtered based on the typed value
  • aria-activedescendant on the input tracks which option is currently focused
  • Each option uses aria-selected to convey its selection state
  • ↑ and ↓ navigate options, Enter selects, Esc closes the dropdown
  • Home and End move focus to the first and last visible option

Single Selection

Selecting an option fills the input with the chosen value and closes the dropdown.

Multi Selection

aria-multiselectable="true" on the listbox communicates that multiple options can be selected. Selecting an option keeps the dropdown open so the user can continue choosing.

Radio Button

Radio buttons allow users to select exactly one option from a group. They are suited for situations where the options are mutually exclusive and all choices should be visible at once.

Accessibility Features

= Required = Recommended
  • <fieldset> and <legend> group the radio buttons and provide a label for the entire group, announced by screen readers before each option
  • name attribute groups radio buttons so that only one can be selected at a time
  • <label> associates visible text with each individual radio button
  • type="radio" provides the native radio role and communicates selected state without additional ARIA

Spinbutton

A spinbutton is a numeric input that allows users to select a value from a defined range. Users can type a value directly or adjust it using the increment and decrement controls.

Spinbuttons are suited for numeric values within a constrained range, such as quantities or counts. For wider ranges where approximate values are acceptable, a slider may be more appropriate.

Accessibility Features

= Required = Recommended
  • <label> associates the visible label with the input so screen readers announce the field name when it receives focus
  • min attribute defines the lower limit and is communicated to screen readers
  • step sets the increment amount when using the arrow keys
  • The + and − buttons can be hidden from keyboard and screen readers with aria-hidden="true" and tabindex="-1", since the input handles keyboard interaction natively. If left accessible, they require descriptive aria-label attributes and a visually hidden aria-live="polite" region that announces the updated value to screen readers when the + and − buttons are pressed

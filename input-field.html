Input Field

An input field is a form control that allows users to enter or select a value. They are a core component of forms, used wherever the interface needs to collect data from the user.

Text Input

A single-line text field that allows users to enter freeform text, such as a name, email address or search query.

Checkbox

A checkbox allows users to select or deselect an option. A switch is a binary toggle that turns a setting on or off.

Radio Button

Radio buttons allow users to select exactly one option from a group of mutually exclusive choices.

Slider

A slider allows users to select a numeric value within a defined range by dragging a handle along a track.

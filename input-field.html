Input Field

An input field is a basic form control that allows users to enter or select a value. They are a core component of forms, used wherever the interface needs to collect data from the user.

Text Input

A text input is a basic form control that allows users to enter freeform text.

Checkbox

A checkbox allows users to select or deselect an option. It can also be applied to create an on/off switch.

Spinbutton

A spinbutton is a numeric input that lets users type a value or adjust it with increment and decrement controls.

Slider

A slider allows users to select a numeric value within a defined range by dragging a handle along a track.

Accessible HTML Components

Accordion

An accordion allows users to expand and collapse sections of content as needed.

Structure/Layout

Alert

An alert is a message that communicates important feedback to the user in response to an action or system event.

Notification

Button

A button triggers an action or event when activated.

Function/Input

Breadcrumb

A breadcrumb trail consists of a list of links to the parent pages of the current page in hierarchical order.

Navigation

Card

A card is a contained unit that groups related information and actions about a single subject.

Structure/Layout

Carousel

A carousel cycles through several content panels in the same area of the page.

Structure/Layout

Combobox

A combobox is an input widget that has an associated dropdown list of options.

Function/Input

Dialog

A dialog is a window overlaid on the page that interrupts the user's flow to request input or display important information.

Notification

Input Field

An input field is a form control that allows users to enter or select a value.

Function/Input

Input Group

An input group is a set of related form controls grouped together under a shared label.

Structure/Layout

Link

A link navigates the user to another location.

Navigation

Landmarks

Landmarks are used to designate major sections of a webpage.

Structure/Layout

Menu

A navigation menu lists the main sections of a website and lets users move between them.

Navigation

Pagination

Pagination allows users to navigate between pages of content, one page at a time.

Navigation

Stepper

A stepper guides users through a multi-step process by displaying all steps and indicating progress.

Navigation

Table

A table organises data into rows and columns.

Structure/Layout

Tabs

Tabs organise content into multiple panels, of which only one is visible at a time.

Navigation

Tooltip

A tooltip is a short text label that provides supplementary information about an element.

Notification

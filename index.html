Accessible HTML Components

Accordion

An accordion allows users to expand and collapse a section of content as needed.

Structure/Layout

Alert

An alert is a message that communicates important feedback to the user in response to an action or event.

Notification

Button

A button triggers an action or event when activated.

Function/Input

Breadcrumb

A breadcrumb trail consists of a list of links to the parent pages of the current page in hierarchical order.

Navigation

Cards

Cards are containers that group related information and actions about a single subject into a compact block.

Structure/Layout

Carousel

A carousel cycles through a sequence content panels within a single compact section.

Structure/Layout

Combobox

A combobox is an input control that displays a dropdown list of options.

Function/Input

Dialog

A dialog is a window overlaid on the page that interrupts the user's flow to request input or display important information.

Notification

Form

A form is a section containing interactive controls that collect input from the user.

Structure/Layout

Input Field

An input field is a form control that allows users to enter or select a value.

Function/Input

Link

A link navigates the user to another location.

Navigation

Landmarks

Landmarks are used to designate major sections of a webpage.

Structure/Layout

Menu

A navigation menu lists the links to the main pages of a website.

Navigation

Pagination

Pagination allows users to navigate between pages of content, one page at a time.

Navigation

Stepper

A stepper guides users through a multi-step process by displaying all steps and indicating progress.

Navigation

Table

A table organises data into rows and columns.

Structure/Layout

Tabs

Tabs organise content into multiple panels, of which only one is visible at a time.

Navigation

Tooltip

A tooltip is a small popup that provides supplementary information about an element.

Notification

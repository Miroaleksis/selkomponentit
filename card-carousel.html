Card Carousel

A card carousel displays several cards side by side and scrolls horizontally as the user moves focus between them.

Card carousels can be used when several similar items can be browsed at a glance, such as articles, products, or related content. They are less suitable when each item needs the user's full attention or when the items are not easily comparable.

Accessibility Features

= Required = Recommended
  • The carousel is wrapped in role="region" with aria-roledescription="carousel" and an aria-label, so screen readers announce its purpose
  • Cards include focusable interactive elements (links or buttons), so keyboard and screen reader users can reach every card with Tab
  • The browser scrolls the focused card into view automatically when an off-screen card receives focus — no JavaScript is needed for this behaviour
  • A skip link before the carousel lets keyboard users bypass all cards and jump directly to the content that follows
  • scroll-snap-type aligns cards to a consistent position after scrolling

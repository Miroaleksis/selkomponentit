Slider

A slider allows users to select a numeric value within a defined range by dragging a handle along a track. It is well suited for settings where the precise value matters less than the relative position, such as volume, brightness or price range.

The native <input type="range"> element should always be preferred. It provides full keyboard support and screen reader announcements out of the box.

Accessibility Features

= Required = Recommended
  • The slider must have a visible <label> associated via matching for and id attributes
  • min, max and step attributes define the allowed range and increment
  • ← → arrow keys decrease and increase the value by one step
  • The current value is displayed visibly next to the slider and updates as the user moves the handle

Multi-Thumb Slider

A multi-thumb slider uses two handles to define a range, such as a minimum and maximum price. Each thumb must have its own <label>, and both thumbs must be grouped with a <fieldset> and <legend>.

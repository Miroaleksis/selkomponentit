Menu

A navigation menu lists the main sections of a website and lets users move between them. Some items can expand into submenus that group related links.

Navigation menus can be used in the site header to provide consistent access to the most important pages. They are not suited for in-page actions or commands. Use buttons or a dialog for those.

Accessibility Features

= Required = Recommended
  • The menu is wrapped in <nav> with an aria-label, so screen readers announce it as a navigation landmark
  • aria-current="page" on the active link identifies the current page to screen readers
  • aria-expanded on the submenu button reflects whether the submenu is currently expanded
  • An expanded submenu can be dismissed with Esc

Without Parent Links

In this variant, items with submenus use a button as the trigger. Clicking the button expands or collapses the submenu.

With Parent Links

In this variant, items with submenus have both a link to the parent page and a separate button that toggles the submenu. On wider screens the submenu also opens on hover. Hover-opened submenus can be dismissed with Esc.

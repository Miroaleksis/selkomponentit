Tooltip

A tooltip is a small popup that provides supplementary information about an element. It appears near its trigger and stays hidden until activated.

Tooltips can be used to clarify the meaning of terms and abbreviations that benefit from a brief explanation. They are not suited for essential information, as they can be difficult to discover for some users.

Accessibility Features

= Required = Recommended
  • A tooltip is activated when a dedicated trigger element either receives hover or focus, or is pressed. Using a pressable trigger button is more reliable than solely hover- or focus-based activation, which can behave inconsistently on touchscreens
  • aria-expanded on the trigger button communicates the expanded or collapsed state to screen readers
  • It a tooltip obscures content, it must be must be dismissable with Esc
  • aria-controls on the trigger button should point to the tooltip
  • role="tooltip" should be used to identify the tooltip
  • The tooltip should be dismissed by pressing the button again or clicking outside

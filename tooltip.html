Tooltip

A tooltip is a short text label that provides supplementary information about an element. It appears near its trigger and stays hidden until activated.

Tooltips can be used to clarify the purpose of icon buttons, abbreviations, or other interface elements that benefit from a brief explanation. They are not suited for essential information, as they can be difficult to discover for some users.

Accessibility Features

= Required = Recommended
  • role="tooltip" on the tooltip element identifies it to assistive technologies
  • Tooltip can be expanded by either hovering or focusing on the trigger element or by pressing Space or Enter
  • If the tooltip is expanded by pressing the toggle button, its state should be indicated using aria-expanded
  • The tooltip can be dismissed with Esc without moving hover or focus (WCAG 1.4.13)

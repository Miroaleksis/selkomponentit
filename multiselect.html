Multiselect

A multiselect combobox allows users to choose multiple options from a dropdown list simultaneously. It can be used whenever users must choose from a known, limited set of values where more than one option applies, similarly to a checkbox group.

The native <select multiple> element provides a similar function as a multiselect combobox but renders as a scrollable listbox rather than a dropdown, so it is not demonstrated here.

Accessibility Features

= Required = Recommended
  • A multiselect combobox has a similar structure and keyboard behaviour as the select-only combobox apart from a few exceptions:
  • aria-multiselectable="true" on the listbox communicates that multiple options can be selected
  • Each option uses aria-selected="true/false" to convey its selection state
  • Space and Enter toggle selection of the focused option without closing the dropdown
  • aria-activedescendant on the trigger tracks which option currently has focus
  • Trigger label updates to reflect the current selection

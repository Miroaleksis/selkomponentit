Multiselect

A multiselect combobox is an input widget that allows users to choose one or more options from a dropdown list. Unlike the Select-Only variant, multiple options can be selected simultaneously.

Multiselect comboboxes are suitable when users need to filter by several categories, assign multiple tags, or pick from a known set of values where more than one answer applies.

The native <select multiple> element provides a similar function but renders as a scrollable listbox rather than a dropdown, so it is not demonstrated here.

Accessibility Features

= Required = Recommended
  • Options are navigable with ↑ and ↓ arrow keys
  • Space and Enter toggle selection of the focused option without closing the dropdown
  • Esc closes the dropdown and returns focus to the trigger
  • aria-multiselectable="true" on the listbox communicates that multiple options can be selected
  • Each option uses aria-selected="true/false" to convey its selection state
  • The expanded/collapsed state is conveyed with aria-expanded on the trigger
  • aria-activedescendant on the trigger tracks which option currently has focus
  • Trigger label updates to reflect the current selection
  • Home and End move focus to the first and last option

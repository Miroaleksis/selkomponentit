Link

A link navigates the user to another location, either within the same page or to an external resource. Links and buttons are often confused but serve different purposes. A button performs an action on the current page, while a link navigates to a new location. See the Button page for more information.

Accessibility Features

= Required = Recommended
  • Link text must describe the destination clearly enough to make sense when read out of context
  • Links that open in a new tab must inform users — either in the visible link text or via visually hidden text — to avoid disorientation

Inline Link

An inline link is embedded within a sentence or paragraph. The linked text should be a meaningful phrase within the sentence, not generic text such as "click here" or "read more". Links must be distinguishable from surrounding text by more than color alone. An underline is the most reliable indicator.

External Link

An external link opens a resource outside the current site, typically in a new tab. Users must be informed of this behavior via visible text or a visually hidden alternative. A visual indicator such as an arrow icon helps sighted users identify external links.

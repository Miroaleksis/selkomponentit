Link

A link navigates the user to another location within the same site or page, or to an external resource.

Links and buttons are often confused but serve different purposes. A link navigates the user to a new location. A button performs an action on the current page (See the Button page for more information).

Accessibility Features

= Required = Recommended
  • Link text must describe the destination clearly enough to make sense when read out of context

Inline Link

An inline link is embedded within a sentence or paragraph. The linked text should be a meaningful phrase within the sentence, not generic text such as "click here" or "read more". Links must be distinguishable from surrounding text by more than color alone. An underline is the most reliable indicator.

External Link

An external link opens a resource outside the current site, typically in a new tab. Users should be informed of this behavior via visible text or a visual indicator such as an arrow icon. If a visual indicator is provided, an accessible alternative, such a visually hidden text, is required.

Link

A link navigates the user to another location, either within the same page or to an external resource. Link text should clearly describe the destination without relying on surrounding context.

Accessibility Features

= Required = Recommended
  • Link text must describe the destination clearly enough to make sense when read out of context
  • Links that open in a new tab must inform users — either in the visible link text or via visually hidden text — to avoid disorientation
  • rel="noopener noreferrer" should be used with target="_blank" to prevent the new page from accessing the opener window
  • External links benefit from a visual indicator so sighted users also know the link opens outside the current site

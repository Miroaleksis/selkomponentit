Breadcrumb

A breadcrumb trail consists of a list of links to the parent pages of the current page in hierarchical order. It helps users understand their location within a site and navigate back to previous levels.

Breadcrumbs can be used on sites with deep, hierarchical structures where users may arrive at a page via search or external links. They are less useful on flat sites where the main navigation already covers all pages.

Accessibility Features

= Required = Recommended
  • Wrapped in a <nav> landmark with aria-label="Breadcrumb" to identify the navigation region
  • The current page is marked with aria-current="page" on the last item
  • Uses an ordered list (<ol>) to convey the hierarchical structure to screen readers
  • Separators are purely decorative and hidden from screen readers with content: '…' / ''
